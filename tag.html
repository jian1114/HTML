태그 종류

<html></html> : 문서의 시작과 끝 <head></head> : 문서 자체의 정보들 <body></body> : 문서의 내용(출력되는 내용들) <!doctype html> : 문서 형식을 나타낸다. <meta charset="utf-8"> : Internet이 한국어를 알아들을 수 있게한다. <strong></strong> : 굵은 글씨 <u></u> : 밑줄 글씨 <title></title> : 문서의 이름(웹사이트의 이름) <h1></h1>, <h2></h2>, ... : 문서의 제목 <a href=""></a> : 하이퍼링크 <a href="" target="_blank"></a> : 링크를 새로운 탭으로 연다. <a href="" target="" title=""></a> : 마우스를 링크에 가져다대면 어떤 링크인지 설명하는 문구 <ul><li></li></ul> : 목차 <ol><li></li></ol> : 목차(번호) <p></p> : 문단 표시 - 검색엔진에서 더 효율적이다. <p style="margin-top: px"></p> : 윗문단과의 사이 간격 <br/> : 단순 줄바꿈 - 시와 같이 가독성을 높일 때 효율적이다. <img src=""> : 다운받은 이미지 문서 <img src="" width="%"> : 사진 크기
